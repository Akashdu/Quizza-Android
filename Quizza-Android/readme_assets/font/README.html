Quizza - The FBLA Quiz App

BookKeeper was designed for the management of large scale libraries from the ground up.

Run Instructions

  1. Copy all files to the computer.
  2. Open the BookKeeper directory.
  3. Run BookKeeper.exe
  4. Load Sample Data from File Menu (File > Load Sample Data)
  5. For more details view Documentation (Help > Documentation)
  6. For details about the current view select the question mark at the top right of the view

Location
You can view the source code in the following locations:
  • View the source code in the 'source' folder found in the root directory of the submission
Source Structure
  • The web API source is stored in source\quizza-api
  • The android app source is stored in source\quizza-android

Features

  • Creatable user accounts
  • Online and offline quiz creation and battles
  • Editable quizzes with an in-app quiz editor
  • Instant feedback on correct and incorrect answers
  • An overview screen after finishing a quiz, displaying all questions and answers
  • Easy bug reports in case the user notices a bug with the application

Usage

Quizza can be installed on an Android Device (Android 5.1 and up) by using the provided .apk file.

Generating Application Binaries (.apk)

./gradlew :base:bundleDebug

Applications can also be generated using Android Studio by opening the project file (Quiz.iml or build.gradle)
